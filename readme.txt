== Description ==

Access control plugin for custom websites.

*   Control access to custom fields
*   Control access to types plugins
*   Control access to "Plugins" and "Themes"

== Installation ==

The quickest method for installing the importer is:

1. Visit Plugins -> Add New in the WordPress dashboard
2. Search for "Omak Access"
3. Click "Install Now"
4. Finally click "Activate Plugin"

If you would prefer to do things manually then follow these instructions:

1. Upload the `omak-acess` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit settings from the Dashboard Options page "Slick Popup"
